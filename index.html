API Coverage By Swagger

Понимайте, какие эндпоинты
уже под покрытием

Инструмент показывает, на какие эндпоинты при тестировании вы уже обратили внимание, а какие остались без покрытия.

Как работает
1
Укажите базовый URL сервиса и ссылку на OpenAPI / Swagger JSON-схему.
2
Запустите ваши тесты, указав в качестве base URL адрес этого сервиса.
3
Обновите страницу — инструмент покажет, какие эндпоинты покрыты тестами, а какие ещё нет.
